YouTube Shorts
  1. Go to Google Cloud Console
  2. Create a project → Enable YouTube Data API v3
  3. Go to Credentials → Create OAuth 2.0 Client ID (Web Application)
  4. Add redirect URI: http://localhost:3000/api/auth/callback/youtube
  5. Copy Client ID & Secret into your .env file
Instagram Reels
  1. Go to Meta Developer Portal
  2. Create an App → Add Instagram Graph API product
  3. Add redirect URI: http://localhost:3000/api/auth/callback/instagram
  4. Copy App ID & Secret into your .env file
  5. ⚠️ Requires an Instagram Business/Creator account linked to a Facebook Page
